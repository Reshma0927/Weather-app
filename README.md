# 🌦 Weather App

A responsive Weather App built using React and Axios that fetches real-time weather information using the OpenWeatherMap API.

## Features

- Search weather by city name
- Displays:
  - City Name
  - Temperature
  - Weather Condition
  - Humidity
  - Wind Speed
- Responsive design
- Clean and modern UI
- API integration using Axios

## Tech Stack

- React.js
- JavaScript
- Axios
- CSS3
- OpenWeatherMap API

## Installation

1. Clone the repository

```bash
git clone https://github.com/Reshma0927/weather-app.git
```

2. Navigate to the project

```bash
cd weather-app
```

3. Install dependencies

```bash
npm install
```

4. Replace :

```bash
YOUR_API with your API key
```

5. Start the development server

```bash
npm run dev
```
